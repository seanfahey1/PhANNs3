# PhANNs

PhANNs is a tool to classify any phage ORF by protein class. It uses an ensemble of Artificial Neural Networks.

This app is a work in progress. 

## Paper

PhANNs paper is available [here](https://journals.plos.org/ploscompbiol/article?id=10.1371/journal.pcbi.1007845).
